OnlyCats Mobile — Cat Rescue & Adoption (Admin Panel)

Flutter · Firebase · Admin Dashboard · Mobile App

OnlyCats adalah platform mobile untuk mendukung rescue, perawatan, dan adopsi kucing jalanan. Aplikasi ini memiliki dua sisi: User untuk pelaporan rescue dan pengajuan adopsi, serta Admin Panel yang menjadi fokus kontribusi saya — sebuah dashboard lengkap untuk mengelola seluruh operasional platform.

Fitur Admin Panel (My Role)

Saya bertanggung jawab penuh atas pengembangan Admin Panel — mulai dari desain UI di Figma hingga implementasi di Flutter dengan integrasi Firebase. Berikut fitur-fitur yang saya bangun:

1. Dashboard Utama & Insights

Halaman utama admin menampilkan ringkasan statistik (total kucing, laporan rescue, adopsi) beserta navigasi ke seluruh menu administrasi. Halaman Insights menyediakan data analitik operasional secara real-time dari Firestore.

Dashboard Utama — Menu Administrasi
Insights — Analitik & Status Operasional
Ringkasan statistik: Total Kucing, Laporan Rescue, Total Adopsi, Adopsi Pending
Status operasional rescue: Menunggu, Sedang Diproses, Telah Selesai
Aktivitas terkini dari Firestore real-time stream
Navigasi cepat ke semua modul administrasi

2. Manajemen Laporan Rescue

Admin dapat melihat dan mengelola semua laporan rescue yang masuk. Laporan dapat difilter berdasarkan status (Semua / Menunggu / Diproses / Selesai) dan diperbarui statusnya langsung dari daftar.

Laporan Rescue — Filter & Kelola Status
Notifikasi — Rescue Baru & Adopsi Masuk
Tab filter: Semua, Menunggu, Diproses, Selesai
Setiap laporan menampilkan foto, lokasi, kondisi kucing, nomor HP pelapor
Update status rescue langsung dari card (dropdown)
Notifikasi push otomatis saat laporan rescue baru masuk

3. Inventory Kucing (CRUD)

Admin dapat mengelola data kucing yang tersedia untuk adopsi. Tersedia fitur tambah, edit, dan hapus data kucing beserta upload foto ke Firebase Storage dan pemilihan lokasi via peta.

Daftar Kucing
Tambah Kucing
Edit Kucing
Daftar kucing dengan nama, ras, umur, lokasi, dan status ketersediaan
Form tambah/edit: upload foto, nama, ras, umur, lokasi, warna, berat
Integrasi Open Street Maps untuk pilih titik lokasi via peta interaktif
Upload foto ke Firebase Storage dengan preview langsung
Hapus data kucing dengan konfirmasi

4. Kelola Adopsi

Halaman ini menampilkan semua formulir adopsi yang masuk dari user. Admin dapat menyetujui atau menolak pengajuan adopsi, dengan filter berdasarkan status.

Kelola Adopsi — Approve / Reject
Tab filter: Semua, Menunggu, Disetujui, Ditolak
Data pengaju: nama, kucing yang diminta, lokasi, nomor telepon
Tombol approve/reject dengan update status real-time ke Firestore
Notifikasi otomatis dikirim ke user saat status berubah

5. Chat Masuk (Admin–User)

Fitur chat real-time memungkinkan admin berkomunikasi langsung dengan user. Admin melihat semua percakapan aktif dalam satu daftar dan dapat membalas pesan secara langsung.

Daftar Chat Masuk
Detail Percakapan
Daftar semua percakapan dari user yang aktif beserta pesan terakhir
Real-time messaging dengan Cloud Firestore stream
Bubble chat dibedakan: pesan user (kiri) dan balasan admin (kanan)
Notifikasi push saat ada pesan baru dari user

6. Manajemen Akun Pengguna

Admin dapat melihat dan mengelola seluruh pengguna terdaftar, dengan fitur pencarian berdasarkan nama atau email, serta hapus akun pengguna.

Daftar Pengguna Terdaftar
Pencarian real-time berdasarkan nama atau email
Tampil: nama, email, username, dan bio pengguna
Hapus akun pengguna dari Firebase

7. Pengaturan & Profil Admin

Halaman akun admin menyediakan pengaturan profil pribadi dan keamanan akun.

Halaman Akun Admin
Edit Profil Admin
Ganti Password
Edit profil: nama lengkap, username, bio, foto profil
Ganti password dengan verifikasi password lama (maks. 5 percobaan)
Pengaturan notifikasi push
Tentang Aplikasi (v1.0.0) dan tombol Keluar

Kesimpulan

Dalam project OnlyCats Mobile, saya membangun seluruh sisi Admin Panel — mencakup 7 modul fungsional yang terintegrasi penuh dengan Firebase. Pengerjaan meliputi desain UI di Figma, implementasi Flutter dengan StatefulWidget, real-time data sync via Firestore, manajemen file di Firebase Storage, dan sistem notifikasi lokal. Hasil akhir adalah admin panel yang fungsional, responsif, dan siap digunakan untuk operasional rescue & adopsi kucing.
