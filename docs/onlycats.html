About the Project
Course
Web Programming

OnlyCats is a web-based information system designed to bridge the process of rescuing and adopting stray cats. It supports three main roles: Visitor (view general info), User (report rescues & apply for adoption), and Admin (manage data & verify requests).

My Contribution
Adoption Feature
My Role — Adoption Feature

I was responsible for the entire Adoption feature flow, from the user side to the admin side. This includes adoption requests, admin verification, real-time status updates, and the user's adoption history page.

1
Adoption Catalog (User) — Users browse the list of available cats on the Adopt page, with photos and basic cat information.
2
Adoption Request (User) — Users select a cat and fill in an adoption request form with their reason (Reason/Notes). Data is saved to the database with a pending status.
3
Admin Verification — Admin receives all requests on the Adoption Requests page. Admin can view cat & adopter details, then choose to Approve or Reject with an optional note.
4
Real-Teame Status (User) — The admin's decision instantly appears on the user's My Adoption page. Status updates to APPROVED (green) or REJECTED (red) along with the admin's note.
5
Admin Dashboard Stats — Approved Adoptions & Pending Adoptions statistics are automatically updated on the Admin Dashboard as a system data summary.
